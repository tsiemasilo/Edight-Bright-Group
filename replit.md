# EDIGHT (PTY) - EDGAR BRIGHT GROUP Website

## Overview
A stunning, professional website for EDGAR BRIGHT GROUP, a visionary IT solutions company based in Pretoria, Gauteng, South Africa. The website features advanced scroll animations, modern design, and comprehensive information about the company's services, training programs, and mission.

## Project Structure
- **5 Pages**: Home, About, Services, Training, Contact
- **Advanced Animations**: Intersection Observer, scroll-triggered reveals, slide animations, counter animations, stagger effects
- **Theme Colors**: Sophisticated palette with Beige (#D1C7C1), White (#FFFFFF), multiple Gray shades (#171717-#F5F5F5), and Black (#000000)
- **Design**: Premium aesthetic with glassmorphism, high-contrast gradients, modern typography, and elegant visual effects
- **Technology**: Pure HTML, CSS, JavaScript - no frameworks for optimal performance

## Technical Details
- **Frontend**: Vanilla JavaScript with IntersectionObserver-based scroll animations
- **Server**: Node.js HTTP server on port 5000
- **Host**: 0.0.0.0 for Replit proxy compatibility
- **CSS Features**: CSS custom properties, backdrop-filter glassmorphism, gradient overlays, clamp-based responsive typography
- **Deployment**: Configured for Replit autoscale deployment

# EDIGHT (PTY) - EDGAR BRIGHT GROUP Website

## Overview
A stunning, professional website for EDGAR BRIGHT GROUP, a visionary IT solutions company based in Pretoria, Gauteng, South Africa. Completely redesigned with React + TypeScript + Tailwind CSS featuring advanced animations, modern design, and comprehensive information about the company's services, training programs, and mission.

## Project Structure
- **Tech Stack**: React 18 + TypeScript + Vite + Tailwind CSS
- **5 Pages**: Home, About, Services, Training, Contact
- **Advanced Animations**: Framer Motion with scroll reveals, slide animations, fade effects, stagger effects, and floating elements
- **Theme Colors**: 
  - Primary: Blue (#0ea5e9 - #0c4a6e)
  - Accent: Orange (#f97316 - #7c2d12)
  - Neutrals: Beige (#fdfcfb - #5e5447), Gray (#f5f5f5 - #171717), White (#FFFFFF)
- **Design**: Premium modern aesthetic with gradient backgrounds, glassmorphism navbar, smooth animations, modern typography
- **Icons**: Lucide React icons throughout
- **Fonts**: Inter (sans-serif) + Playfair Display (display/headings)

## Technical Details
- **Frontend**: React 18.3.1 with TypeScript 5.6.3
- **Build Tool**: Vite 5.4.21 for fast HMR and optimized production builds
- **Styling**: Tailwind CSS 3.4.15 with custom animations and color palette
- **Animations**: Framer Motion 11.11.11 for smooth, performant animations
- **Routing**: React Router DOM 6.28.0 with future flags enabled
- **Server**: Vite dev server on port 5000 (0.0.0.0 for Replit compatibility)
- **Deployment**: Configured for Replit autoscale deployment

## How to Run
The project runs automatically via the configured workflow. The Vite dev server starts on port 5000 with:
```bash
npm run dev
```

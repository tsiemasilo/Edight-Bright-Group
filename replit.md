# Edight Bright Group Website

## Overview
This is a clean, modern website for Edight Bright Group. The project was imported from an empty GitHub repository and initialized with a basic static website structure.

## Project Structure
- `index.html` - Main HTML page with sections for Home, About, Services, and Contact
- `style.css` - Styling with gradient theme and responsive design
- `script.js` - JavaScript for smooth scrolling and form handling
- `server.js` - Simple Node.js HTTP server for serving static files
- `package.json` - Node.js project configuration

## Technical Details
- **Frontend**: Pure HTML, CSS, and JavaScript (no frameworks)
- **Server**: Node.js HTTP server running on port 5000
- **Host**: Configured for 0.0.0.0 to work with Replit's proxy
- **Cache Control**: Disabled caching for immediate updates in development

## Recent Changes
- 2024-11-18: Initial project setup with basic website structure

## How to Run
The project runs automatically via the configured workflow. The server starts on port 5000 and serves the static website files.
